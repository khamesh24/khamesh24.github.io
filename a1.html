Circuit Overview

I built a parallel circuit with three LEDs (Red, Blue, Yellow). Each LED has its own resistor, and its anode connects to a different Arduino digital pin so each LED can blink independently. The cathodes are all connected to ground (GND).

  • Red LED → resistor → Arduino pin 13
  • Blue LED → resistor → Arduino pin 12
  • Yellow LED → resistor → Arduino pin 8
  • All LED cathodes → GND

Circuit Schematic

Each LED has its own resistor to limit current. The anodes connect to digital pins 13, 12, 8, and the cathodes connect to GND.

Resistor Calculations (Ohm’s Law)

Supply voltage (VS) = 5V
Recommended current (I) = 16 mA = 0.016 A
R = V / I, where V is the voltage across the resistor.

Red and Yellow LEDs

  • Forward voltage (VF) = 1.8V
  • Voltage across resistor = 5 - 1.8 = 3.2V
  • R = 3.2 / 0.016 = 200 Ω
  • Used 220 Ω (common value, slightly safer and limits current a bit more)

Blue LED

  • Forward voltage (VF) = 3.3V
  • Voltage across resistor = 5 - 3.3 = 1.7V
  • R = 1.7 / 0.016 = 106.25 Ω
  • Used 120 Ω (common value)

Arduino Code (Each line explained above it)


// the setup function runs once when you press reset or power the board
int redPin = 13;
int bluePin = 12;
int yellowPin = 8;

void setup() {
  // set the red LED pin as an output
  pinMode(redPin, OUTPUT);
  // set the blue LED pin as an output
  pinMode(bluePin, OUTPUT);
  // set the yellow LED pin as an output
  pinMode(yellowPin, OUTPUT);
}

void loop() {
  // turn on the yellow LED
  digitalWrite(yellowPin, HIGH);
  // wait for 100 milliseconds
  delay(100);
  // turn off the yellow LED
  digitalWrite(yellowPin, LOW);
  // wait for 100 milliseconds
  delay(100);

  // turn on the yellow LED again
  digitalWrite(yellowPin, HIGH);
  // wait for 100 milliseconds
  delay(100);
  // turn off the yellow LED
  digitalWrite(yellowPin, LOW);
  // wait for 300 milliseconds before the next LED
  delay(300);

  // turn on the red LED
  digitalWrite(redPin, HIGH);
  // wait for 100 milliseconds
  delay(100);
  // turn off the red LED

Additional Questions

2) How many LEDs could you blink independently, and how much current?

I was able to get 13 LEDs to blink independently. Each draws around 16 mA, so the total ideal current would be 13 × 16 mA = 208 mA. Since the Arduino Uno can safely source about 200 mA total, the current would distribute roughly evenly—around 15.4 mA per LED.

3) How many if only one LED is on at once?

If only one LED is on at a time, the number of independently controlled LEDs is limited by available pins—about 20 (using D2–D13 and A0–A5). The current draw would then just be that of a single LED (about 10–16 mA).

4) Blink speed threshold (persistence of vision)

Around a 10 ms delay per state (100 Hz) the blinking appears steady to the human eye. Increasing the delay to around 15 ms makes the blinking visible again.
